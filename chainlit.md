# Fiducia

Fiducia is a Chainlit-based interface for retrieval-augmented question answering over uploaded document collections.

## What you can do

- Upload document collections and manage indexed files.
- Ask questions across a selected collection or all collections.
- Inspect timelines extracted from document content.
- Generate cached facts summaries for collections.
- Open source PDFs in the built-in sidebar viewer.

## Notes for PoC users

- This project depends on external services (Qdrant and model providers).
- OAuth is integrated; configure auth variables before startup.
- See `README.md` for quickstart and `docs/` for operational details.
